Framework & Tool STABLE

MOD
LOADER

Release v1.3.0 — UE4SS / Lua

A mod compatibility framework for Mad Television Tycoon. Injects entries from mod JSON files into the vanilla database without touching or rewriting vanilla content — letting any number of mods coexist without conflicts. Supports items, cast, names, reviews, and GMPL files.
