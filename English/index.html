📖 How to Install Your Mod

1

Install the Modloader

The Modloader is a small program that lets Mad TV Tycoon read your custom files. Without it, the game ignores everything in the Mods folder.

⬇ Download the Modloader — link coming soon! Check the game's official community page for the latest version.

2

Create your content in this tool

Use the tabs on the left to add movies, TV shows, cast members, and more. Don't worry about IDs or technical stuff — this tool handles all of that for you.

Tip: Set a unique Mod Name in the top bar (e.g. your username or mod title). This is used in your filenames to make sure your mod doesn't clash with other mods.
3

Export your mod files

Click Export Mod in the bottom-left. Each file type gets its own download button. Only download the files for content you actually added.

You added...File to downloadExample filename
Movies / TV Series / Shows_items.jsonmymods_items.json
Cast members_cast.jsonmymods_cast.json
Random name lists_names.jsonmymods_names.json
Review texts_reviews.jsonmymods_reviews.json
4

Drop the files into the Mods folder

Open your Mad TV Tycoon installation folder. Inside it, there should be a folder called Mods. Simply drag and drop your downloaded JSON files into that folder.

Where is the installation folder?
On Steam: right-click the game → Manage → Browse local files
You'll see a Mods folder inside.
5

Start the game and enjoy!

Launch Mad TV Tycoon normally. If the Modloader is installed and your files are in the Mods folder with the right names, your new content will appear in the game automatically.

Something not working? Make sure:
✅ The Modloader is installed
✅ Your filename follows the pattern: yourname_items.json (the part after the underscore must be exactly items, cast, names, or reviews)
✅ You only have one file per type with the same prefix name

⬇ Export Mod

✅ Done!
